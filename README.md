# DongDong Guardian (RN)

보호자/환자 앱(guardian) React Native(Expo Router) 프로젝트입니다.

## Docs (현재 기준)

- `docs/README.md`

## Run (DEV)

1) 환경변수 설정(권장: `.env` 또는 쉘 env)

- 참고: `ENV.example.txt`

2) 실행

```bash
npm install
npx expo start
```

> 카카오 로그인 SDK는 **Expo Go에서 동작하지 않습니다**. 개발 빌드/스토어 빌드 기준으로 검증합니다.
